在个人网页上加入网易云音乐的歌曲
发布于：2018-07-28 05:06:41 标签：/ hexo / 访问：

想在博客中加入可以播放的歌曲链接。自行百度后找到了网易云音乐提供的外链播放器功能。

通过网易云音乐网页版，可以生成外链播放器的代码。

不过一些有版权限制的歌曲是不能生成外链播放器播放的哦。

提供了两种形式。

第一种.通过单首歌曲生成

操作过程

在网易云音乐网页版中选择一首歌曲，进入歌曲播放页面。在左边选择生成外链播放器。

然后进入播放器生成页面，配置样式，一键复制代码
