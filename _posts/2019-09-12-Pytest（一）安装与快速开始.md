pytest是一个可以轻松构建简单且可扩展测试用例的框架。本文通过一些小的单元测试和功能测试，来快速了解pytest的使用。 
<!-- more -->

## 安装pytest

1. 运行下面的命令安装pytest模块：
```
>pip install -U pytest # install -U 意思是更新xx模块
```

2. 检查pytest是否安装成功：
```
>pytest --version
```

## 创建第一个test

创建一个文件，包含一个被测试函数和一个测试函数：
```python
# test_first_pytest.py file


# 被测试的函数func(x)
def func(x):
    return x + 1

# 执行测试的函数test_func()
def test_func():
    assert func(1) == 3 

```

然后，在`test_first_pytest.py`文件路径下，执行pytest命令:
```shell
>pytest


```
